# About IoT/Communication Modules

IoT/Communication Modules are core components for achieving device interconnection and data transmission, covering various types such as wireless communication, wired interfaces, and protocol conversion. These modules provide stable data connectivity for various terminal devices, supporting long-range wireless communication, short-range wireless connections, and multiple industrial interface expansions. They are widely used in fields such as smart terminals, industrial control, remote monitoring, and automated systems, providing fundamental support for device network access and data interaction.
