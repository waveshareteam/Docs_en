# IMX415-8MP-USB-Camera-A

<div style={{maxWidth:500}}>
[![IMX415 USB Camera (A) Product Image](https://www.waveshare.net/photo/accBoard/IMX415-8MP-USB-Camera-A/IMX415-8MP-USB-Camera-A-details-1.jpg)](https://www.waveshare.com/imx415-8mp-usb-camera-a.htm)
</div>

The IMX415 8MP USB Camera module is equipped with an IMX415 8MP sensor chip. The module features a 95.5° wide-angle, distortion-free lens with autofocus, dual microphones for USB audio recording. It is equipped with a USB2.0 Type-C port, convenient and easy to connect.
