# PS5268-2MP-USB-Camera-A

<div style={{maxWidth:500}}>
[![PS5268 2MP USB Camera (A) Product Image](https://www.waveshare.com/img/devkit/accBoard/PS5268-2MP-USB-Camera-A/PS5268-2MP-USB-Camera-A-details-1.jpg)](https://www.waveshare.com/ps5268-2mp-usb-camera-a.htm)
</div>

The PS5268-2MP-USB-Camera-A module is based on the PixArt PS5268 sensor and utilizes a USB interface for signal transmission, offering excellent compatibility. Its convenient plug-and-play design allows for direct connection to a host device, simplifying application development. The module delivers a stable 30FPS frame rate, ensuring smooth and fluid video. Furthermore, its 120dB high dynamic range (HDR), combined with exceptional image quality, captures rich image details clearly, performing reliably whether under bright daylight or in low-light nighttime conditions. The PS5268-2MP-USB camera module also features a waterproof housing, enhancing its environmental adaptability. It is suitable for various applications requiring high-precision visual data, such as industrial inspection, robotic vision, and security surveillance, providing accurate and reliable visual data support.
