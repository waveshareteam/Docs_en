# User Guide

This module must be used with a GMSL deserializer board. It supports connection to a Raspberry Pi or Jetson Orin Nano/NX main board. For more information, please refer to the [GMSL Deserializer Board User Guide](/docs/Others/Camera-Audio-Video/GMSL-Deser-Board/MAX9296-GMSL-Deser-Module/Jetson_Orin.md)
