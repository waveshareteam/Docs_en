# FAQ

<Details summary="Q: Driver installation failed, with error messages?" className="faq-details" open>

A: Please check if the system version in use is the compatible kernel version. Please test with the required system version.

</Details>

<Details summary="Q: Driver is installed normally, but the preview displays no image?" className="faq-details" open>

A: Please check if the hardware connections are made to the specified ports, check if the cables are damaged. After re-seating the cables, power cycle the system and test again.

</Details>
