# 7inch 1080x1080 LCD

## Features
- 7inch IPS round screen with 1080x1080 hardware resolution
- 10-point capacitive touch with toughened glass panel, hardness up to 6H
- Uses optical bonding touch process for better display effect
- When used with Raspberry Pi, supports Raspberry Pi OS / Ubuntu / Kali and Retropie
- When used as a computer monitor, supports Windows 11 / 10 / 8.1 / 8 / 7
- Supports HDMI audio output, onboard 3.5mm headphone jack and 4PIN speaker interface
