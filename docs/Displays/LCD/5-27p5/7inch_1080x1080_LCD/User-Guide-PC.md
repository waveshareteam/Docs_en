## Hardware Connection
- 1. Connect the LCD's Touch interface to a USB port on the PC. Windows will automatically recognize the touch function.
- 2. Connect the LCD's HDMI interface to an HDMI port on the PC. Windows will automatically recognize the display function.

:::warning 注意

Some PCs do not support HDMI plug-and-play; usually, restarting the system will enable normal use.                               
If the USB port cannot supply sufficient power, it may cause screen flickering. This can be resolved by connecting an external 5V 1A power adapter to the LCD's Power interface.
:::

## Windows Specified Touch Screen
### Windows11:

1. Open Windows Settings, type and click "Control Panel" in the search bar. 
<div style={{maxWidth:600}}> <img src={Windows11_Touch_1} alt=" 7inch_1080x1080_LCD 硬件连接"/></div>
2. In the "Control Panel" window, select "Hardware and Sound".
<div style={{maxWidth:600}}> <img src={Windows11_Touch_2} alt=" 7inch_1080x1080_LCD 硬件连接"/></div>
3. Under "Calibrate the screen for pen or touch input", in the "Tablet PC Settings" dialog, click "Setup".
<div style={{maxWidth:600}}> <img src={Windows11_Touch_3} alt=" 7inch_1080x1080_LCD 硬件连接"/></div>
4. A text prompt will appear on the screen as shown below. Tap the touch screen with your finger; the computer will then recognize it as the touch screen.
<div style={{maxWidth:600}}> <img src={Windows10_Touch_3} alt=" 7inch_1080x1080_LCD 硬件连接"/></div>

   ```bash
   Note: If the screen is blank, press the "Enter" key; the text prompt will switch to the next screen.
   (The screen displaying the text prompt will be designated as the touch screen.)
   ```

   <div style={{maxWidth:600}}> <img src={Windows10_Touch_1} alt=" 7inch_1080x1080_LCD 硬件连接"/></div>

3. The screen will display a prompt as shown below. Please touch the screen with your finger, and the computer will recognize it as a touch screen.

