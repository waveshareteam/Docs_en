## Hardware Connection

1. Connect the provided 12V 5A power supply to the screen; the backlight will turn on.
2. Connect the LCD's HDMI interface to the PC's HDMI port. Windows will automatically recognize the display function.
3. Connect the LCD's USB interface to the PC's USB port. Windows will automatically recognize the touch function.

:::warning Note

Some PCs do not support HDMI plug-and-play; a system restart usually resolves this.

If the 12V power supply is insufficient, the LCD may flicker. It is recommended to use the included 12V 5A power adapter or manually reduce the brightness to restore normal operation.

:::

## Windows Specified Touch Screen

### Windows11:

1. Open Windows Settings, type and click "Control Panel" in the search bar.
   <div style={{maxWidth:600}}> <img src={Windows11_Touch_1} alt=" 23.8inch FHD Monitor Hardware Connection"/></div>
2. In the "Control Panel" window, select "Hardware and Sound".
   <div style={{maxWidth:600}}> <img src={Windows11_Touch_2} alt=" 23.8inch FHD Monitor Hardware Connection"/></div>
3. Under "Calibrate the screen for pen or touch input", in the "Tablet PC Settings" dialog, click "Setup".
   <div style={{maxWidth:600}}> <img src={Windows11_Touch_3} alt=" 23.8inch FHD Monitor Hardware Connection"/></div>
4. A text prompt will appear on the screen as shown below. Tap the touch screen with your finger; the computer will then recognize it as the touch screen.
   <div style={{maxWidth:600}}> <img src={Windows10_Touch_3} alt=" 23.8inch FHD Monitor Hardware Connection"/></div>

   :::tip
   Note: If the screen is blank, press the "Enter" key; the text prompt will switch to the next screen.
   (The screen displaying the text prompt will be designated as the touch screen.)
   :::
