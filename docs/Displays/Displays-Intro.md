# Displays
Displays are a core component of human-machine interaction systems, responsible for intuitively presenting the device's operating status, image information, and user interface. In the fields of embedded development, industrial control, smart terminals, and consumer electronics, display modules not only handle information output but are also gradually becoming important interactive portals.
