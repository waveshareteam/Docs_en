The ESP32-C3 is a secure, stable, low-power, and low-cost IoT chip equipped with a RISC-V 32-bit single-core processor. It supports 2.4 GHz Wi-Fi and Bluetooth 5 (LE), providing IoT products with industry-leading RF performance, comprehensive security mechanisms, and abundant memory resources. The dual support for Wi-Fi and Bluetooth 5 (LE) in the ESP32-C3 simplifies device provisioning, making it suitable for a wide range of IoT application scenarios.
