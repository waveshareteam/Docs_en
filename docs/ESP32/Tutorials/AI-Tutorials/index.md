# AI Application Tutorials

Welcome to the Waveshare AI Application Tutorials!

This tutorial series is designed to help developers master the quick setup of AI agents on Waveshare's ESP32 series development boards that support AI applications. The tutorials include application guides such as XiaoZhi AI. You can choose the appropriate tutorial based on your specific needs.

- [XiaoZhi AI Application Tutorial](/docs/ESP32/Tutorials/AI-Tutorials/01-xiaozhi.md)
